# KWRocketry 引擎参数规划报告

日期：2026-05-23

本文件是 KWRocketry 第一轮参数规划报告。
这一版只做参数定位，不落地 ModuleManager 补丁。
本次已经同步更新液体发动机和固体推进器参数，下一步可以直接转入实际配置文件编写。

## 1. 本轮参数口径

- 数据依据一：C:\Users\26549\OneDrive\文档\KSP Engine Tweak Chart.xlsx 中的 Engine Database 页面。
- 数据依据二：KWRocketry 原始 cfg。
- 数据依据三：你补充的外观、燃烧室数量、循环类型和包络尺寸说明。
- 本版不建议先动外形尺寸，也不建议先改 rescaleFactor，优先通过参数重定位来补齐系列缺口。
- 液体发动机的真空推重比按照 真空最大推力 / 质量 / 9.80665 计算。
- 固体推进器除了干质量推重比外，额外给出计划装药量和名义燃烧时间，便于后续换算 RF 装药量。

## 3. 现有系列的缺口与重定位

- Wildcat 和 Vesta 这两条线，本轮不再靠数据库重新发散，而是直接围绕你给出的 原力110 / 炽 模板收敛。
- Maverick 系列改成液氧甲烷之后，不该再去复制现有 FFSC 可复用主级机，而更适合拆成两档：1D 负责低空可复用主级，Maverick V 负责 110 吨级左右的 SCC 真空甲烷机。
- 从当前工作区样本看，高性能 FFSCC 甲烷机已经不缺，但 2.5m 包络下、推力约 110 吨级、偏真空、Isp 不必过分激进的 SCC 甲烷机仍然是空位，这更适合 Maverick V。
- Titan 改成氢氧之后，这一版不再走“大喷管高空机”路线，而是反过来收包络：Titan T1 改成 1.875m 的高推重比起飞级单机，Titan V 则作为 5 台 T1 并联、相对偏真空一些的 5m 氢氧 core stage 集群。
- Griffon 系列本轮保持不动，继续作为重型多燃烧室液氧煤油主级线。
- SPS 不再按两个 part 硬拆，而是视为一个模型、一个零件，通过 RF 配置切换成两种甲烷方案；但这两套方案都需要明显提高推力和推重比。

### 固体推进器

- Globe I / V / VI / X / X2 可以整理成轻型到中型 PBAN 固推梯度。
- X5 / X10S / X10L 可以整理成重型 2.5m HTPB 固推梯度。
- 现在真正的缺口不是数值，而是没有 3.75m 主力固推，因此这轮不建议把 2.5m Thor 系列硬拉到 3.75m 级别总冲量。
- Ullage motors 应该明确为分离辅助火箭，而不是普通可节流小固推。

## 6. 这版进入落地前最需要先确认的点

1. Wildcat XR 这一版按 原力110 的真空型来做，默认值是 345s / 1318kN / 0.82t，你如果希望更偏上面级或更偏高空主级，我可以再收一轮。
2. Vesta VR9D 这一版按 炽 的双燃烧室真空型来做，默认值是 366s / 1790kN / 1.08t，这个量级是否符合你对 炽V 的预期。
3. Maverick V 这一版已经把真空比冲从 378s 收回到 370s，其他量级保持在 1100kN / 1.15t 的 SCC 真空机区间，这个收缩幅度是否符合你的预期。
